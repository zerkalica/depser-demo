# depser-demo

Example demo project for [mam depser](https://github.com/zerkalica/yuf/blob/master/depser/README.md)

## Build

From zero:

* `git clone https://github.com/zerkalica/depser-demo`
* `cd depser-demo`
* `git submodule update --init --depth=100`
* `.ci/yuf/depser/depser build`

Build in `./app/-`
